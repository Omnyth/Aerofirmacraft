# Lower Ocean Pre-Noise V11

## Purpose

Test whether assigning `tfc:ocean` below Y=0 before `fillFromNoise` affects TFC terrain behavior.

## Reason

V10 successfully assigned existing `tfc:ocean` to the lower ocean biome cells during the noise-stage transform, but that happens after TFC's `createBiomes` stage.

V11 adds a `createBiomes` return hook that assigns `tfc:ocean` to all biome cells from world bottom through the Y=0 biome cell before `fillFromNoise`.

## Expected Outcome

If later TFC terrain generation reads the section biome palette, this gives it the best chance to see ocean at the lower Y level.

If terrain generation remains unchanged, that confirms TFC's physical terrain/water generation is driven by custom generator data rather than the section biome palette. In that case, the lower ocean must remain block-generated by the AFC transform.

## Current Vertical Identity

- Y <= 0 biome cells: `tfc:ocean`
- Y > 0 carved old-ocean/coastal cells: `aerofirmacraft_terrain:sky_gap`
- Y > 0 preserved cells: original TFC biome

# Sky Gap Biome V7

## Purpose

Add the first Aerofirmacraft-specific biome for old-ocean sky gaps:

`aerofirmacraft_terrain:sky_gap`

## Scope

This branch only adds the biome data resource.

It does not yet rewrite generated biome palettes. That is intentionally deferred to the next branch because biome-palette mutation should be tested separately from the data resource.

## Design

Old TFC ocean and reviewed coastal/ocean-edge biomes should eventually become:

`aerofirmacraft_terrain:sky_gap`

This biome represents open air between floating TFC landmasses and the lower ocean layer.

## Current Behavior

Terrain transformation is still inherited from V6:

- reviewed TFC ocean/coastal biomes become sky gaps
- non-ocean biomes are preserved as floating terrain
- lower ocean/crust blocks are still generated by the transform
- rivers, lakes, dune sea, and oceanic mountain biomes remain preserved

# Sky Gap Biome V8 Assignment

## Purpose

Assign `aerofirmacraft_terrain:sky_gap` to old-ocean/coastal sky-gap biome cells above the lower ocean.

## Scope

This branch keeps V6 terrain behavior and V7 biome data, then adds biome-palette mutation.

## Behavior

- old TFC ocean/coastal columns are still carved into air gaps
- non-ocean columns remain preserved as floating terrain
- lower ocean/crust blocks are still generated by the transform
- sky-gap biome is assigned only above Y=0
- sky-gap biome assignment uses biome-cell/quart resolution, not block resolution

## Implementation Notes

Minecraft biome cells are 4x4x4 block regions.

For each 4x4 horizontal biome cell, V8 assigns `sky_gap` if at least half of the underlying block columns are classified as old-ocean/coastal sky-gap columns.

The lower ocean biome layer is intentionally deferred.

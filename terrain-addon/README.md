# Aerofirmacraft Terrain

NeoForge addon prototype for Aerofirmacraft terrain experiments.

## Current milestone

This project currently only proves that an addon can:

- compile
- load
- log common setup
- log server starting

It intentionally does not modify terrain yet.

## Mod ID

aerofirmacraft_terrain

## Package

com.omnyth.aerofirmacraftterrain

## Target

- Minecraft 1.21.1
- NeoForge 21.1.228
- Java 21
